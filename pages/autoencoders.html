Interpolação via AutoEncoders

17/07/2025

O que é Interpolação de Imagens?

Dado um dataset de imagens, como o MNIST, queremos gerar sequências de imagens que transitam de uma amostra para outra. No nosso caso, cada imagem representa um dígito em escala de cinza, e a "tinta branca" define o desenho do número.

O desafio é gerar essas transições de forma que o desenho permaneça conexo — isto é, que não existam "poças" desconexas de tinta em cantos isolados.

Estrategicamente, utilizamos Autoencoders:

  • Treinamos o modelo para reconstruir imagens do MNIST.
  • Geramos instâncias aleatórias no espaço latente que podem conter poças.
  • Corrigimos manualmente essas poças para criar imagens conectadas.
  • Retreinamos o modelo com essas imagens corrigidas, ensinando-o a produzir desenhos sempre conectos.

O objetivo final é gerar vídeos interpolando duas imagens quaisquer, garantindo que a transição seja visualmente contínua e que a tinta permaneça conectada.

Resumo

Este trabalho faz parte da iniciação científica com bolsa CNPq, sob orientação da prof. Márcia Rosana Cerioli. Aborda como usar autoencoders para gerar transições suaves entre imagens do MNIST, mantendo a conectividade dos dígitos.

A estratégia envolve criar um dataset de imagens conectas corrigidas manualmente, que ensina o modelo a evitar poças desconexas, possibilitando interpolação confiável entre dígitos.
